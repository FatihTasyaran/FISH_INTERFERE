# FISH Documentation

**F**unction Level **I**nspection of **S**cheduling **H**ierarchies

FISH is a profiling framework for ROS 2 systems. It captures LTTng
kernel/userspace traces, NVIDIA GPU profiling data, and system state
snapshots from inside the container where your ROS 2 nodes run.
