## Methodology

1. Fresh fish-r2b-<bench>:latest container per benchmark
2. Mount host /home/tue037807/fish_interfere → /host/fish_src
3. Run setup_fish.sh inside — picks up latest fish_tracepoints/install_fish_tracepoints
   (includes the GenericSubscription patch commit 0743b32 that emits rclcpp_subscription_init
   + rclcpp_subscription_callback_added + rclcpp_callback_register)
4. Rebuild ros2_benchmark + isaac_ros_benchmark in container to compile against patched headers
5. Install missing deps (trtexec, dcgm, mcap) per benchmark
6. Wire datasets + models from /host
7. launch_test the bench script with FISH_ENABLED=1
8. Postprocess trace via fish_compose.ingest_session + extract_container_model
9. Compare actual node counts to expected formula composed from:
   - container_expected() = 4 E/F (ComponentManager + 3 _container svcs)
   - launch_ros_expected() = 6/6 (rclpy launch agent)
   - controller_expected() = 6/6 (rclpy benchmark Controller)
   - data_loader_expected() = 11/11 (VCC1 + 4 user svcs)
   - nitros_node(N_in, N_out, runtime_extra=1) = 7 + 1 + 2·N_in + 2·N_out + 1
   - nitros_playback_node(N_formats) = 7 + 3 + 3·N_formats
   - nitros_playback_node_generic(N_formats) = 7 + 3 + N_formats
   - nitros_monitor_node_nitros_sub() = 11/11 (uses NITROS NEGOTIATED sub)
   - nitros_monitor_node_generic(format) = 10/10 (uses create_generic_subscription)
   - managed_nitros_node(n_in=1, n_out=1) = 11/11 (plain rclcpp Node + ManagedNitrosSub/Pub)
   - transform_listener_impl() = 3/3
   - static_transform_publisher() = 7/7
10. Build A2 spreadsheet via direct Google Sheets API (OAuth):
    REFERENCE + bench_node + expected_vertex + actual_vertex + ros2cli_helpers + per-node tabs
